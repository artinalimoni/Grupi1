# Sistemi i Menaxhimit të Parkimit

## Përshkrimi i Projektit
Ky projekt është një aplikacion i thjeshtë konsolë i zhvilluar në **C++**, i cili simulon menaxhimin e një parkingu.  
Programi mundëson regjistrimin e veturave, shfaqjen e tyre, llogaritjen e pagesës së parkimit dhe kërkimin e veturave sipas targës përmes një menuje interaktive.

## Qëllimi i Projektit
Qëllimi i këtij projekti është të demonstrojë përdorimin praktik të:
- Strukturave (`struct`) në C++
- Funksioneve dhe programimit modular
- Ruajtjes së të dhënave përmes array-ve
- Ndërfaqes me menu për ndërveprim me përdoruesin
- Kontrollit të versioneve me Git dhe GitHub

## Funksionalitetet
- Regjistrimi i veturave në parking
- Shfaqja e veturave të parkuara
- Llogaritja e pagesës së parkimit sipas orëve
- Kërkimi i veturës sipas targës
- Menu interaktive për përdoruesin

## Teknologjitë e Përdorura
- **C++**
- **Git**
- **GitHub**

## Si të ekzekutohet projekti

1. Klononi repository-n:
```bash
git clone https://github.com/username/parking-management-system.git
```
2.	Hyni në folderin e projektit:
```bash
cd parking-management-system
```
3.	Kompiloni programin:
```bash
g++ main.cpp -o parking
```
4.	Ekzekutoni programin:
```bash
./parking
```
